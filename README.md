# Работа с лабораторными в репозитории java-application-development

В репозитории находятся 4 лабораторные работы. Каждая лабораторная — это отдельный Maven-модуль.

## Структура проекта

```
java-application-development
 ├─ pom.xml
 ├─ lab-1
 ├─ lab-2
 ├─ lab-3
 └─ lab-4
```

Каждая папка `lab-X` содержит:

```
src/main/java   — код лабораторной работы  
src/test/java   — тесты для проверки задания
```

---

# 1. Клонирование репозитория

```bash
git clone https://github.com/ferbator/java-application-development.git
cd java-application-development
```

---

# 2. Работа с лабораторной

Каждая лабораторная выполняется **в своей папке** и **в своей ветке**.

Пример:

```
lab-1/src/main/java
```

Создавайте и редактируйте код только внутри соответствующего модуля.

---

# 3. Создание рабочей ветки

Работу необходимо выполнять **в отдельной ветке**, созданной от `master`.

1. Убедитесь, что вы находитесь на ветке `master`:

```bash
git checkout master
```

2. Получите последние изменения:

```bash
git pull
```

3. Создайте свою рабочую ветку:

```bash
git checkout -b lab-1
```

После этого **все изменения выполняйте только в этой ветке**.

Проверить текущую ветку можно командой:

```bash
git branch
```
