# Лабораторная работа 1

В модуле сделана простая банковская система на Java.

- клиенты с builder и признаком сомнительности
- несколько банков через центральный банк
- дебетовые, депозитные и кредитные счёта
- пополнение, снятие, переводы между банками
- транзакции с отменой
- ежедневное начисление процентов и месячная выплата
- месячная комиссия по кредиту при отрицательном балансе
- перемотка времени
- уведомления через console, email, sms, push
- консольное приложение

## Как запустить тесты

Из корня репозитория:

```bash
mvn -pl lab-1 test
```

Сейчас в модуле есть 6 тестов:

- лимит для сомнительного клиента
- проценты по дебетовому счёту
- запрет снятия с депозита до срока
- комиссия по кредитному счёту
- межбанковский перевод и отмена транзакции
- уведомления при смене настроек банка

## Как запустить консольное приложение

Из корня репозитория:

```bash
mvn -pl lab-1 exec:java -Dexec.mainClass=itmo.course.Main
```

## Быстрая ручная проверка

1. Создайте банк
2. Создайте клиента
3. Посмотрите клиентов банка и скопируйте `clientId`
4. Откройте счёт
5. Пополните счёт
6. Сделайте перевод или снятие
7. Посмотрите транзакции и отмените одну из них
8. Перемотайте время на месяц и посмотрите, как изменился баланс

## Что важно по реализации

- сомнительный клиент: нет адреса или паспорта
- лимит для сомнительного клиента задается банком
- депозит можно пополнять, но нельзя снимать до конца срока
- ставка по депозиту фиксируется при открытии по начальной сумме
- дебетовый счёт использует текущую ставку банка
- кредитный счёт может уходить в минус в рамках кредитного лимита
